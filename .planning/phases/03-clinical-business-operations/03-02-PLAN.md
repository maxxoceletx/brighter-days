Create the complete Crisis Protocol SOP using the standard SOP heading structure from 03-RESEARCH.md.

Owner: Valentina Park, MD (clinical lead) with Maxi (CTO) for admin support during crisis events.

This is a single comprehensive SOP document containing all of the following sections:

**PRE-SESSION SAFETY SETUP (data collected at intake -- cross-reference SOP-01):**
Document what must be collected at patient intake for crisis readiness:
- Patient's full physical address (not just city -- needed for 911 dispatch)
- Patient's county of residence (for county-specific crisis routing)
- Primary parent/guardian: name, phone, relationship, authorized for crisis notification
- Secondary emergency contact: name, phone
- Patient's local emergency number if different from 911
- Local hospital / nearest ED to patient's home address

**PARENT PRESENCE BY AGE GROUP (include as guidance table):**
- Under 8: Parent present or nearby; provider confirms parent location at session start
- 8-12: Parent nearby (in home); confirm parent can enter if needed
- 13-17: Start session alone; parent available to join; confirm parent contact at session start
- Source note: AACAP telepsychiatry policy statement 2017; MEDIUM confidence (no single authoritative standard by exact age)

**THREE-TIER CRISIS ESCALATION (core of the SOP):**

TIER 1 -- Passive Suicidal Ideation (no plan, no intent):
- [ ] Administer C-SSRS to document current severity
- [ ] Safety planning conversation (collaboratively build safety plan)
- [ ] Contact parent/guardian to inform (for minors -- always)
- [ ] Schedule follow-up within 48 hours
- [ ] Document in Tebra: C-SSRS score, safety plan, parent notification

TIER 2 -- Active Suicidal Ideation (plan OR intent, not both):
- [ ] Administer C-SSRS
- [ ] Contact parent/guardian immediately (during session if possible)
- [ ] Recommend 988 call or county crisis line with parent/patient
- [ ] Recommend voluntary crisis evaluation at nearest ED
- [ ] Document: C-SSRS, clinical reasoning, referrals made
- [ ] Follow-up within 24 hours

TIER 3 -- Imminent Risk (plan AND intent AND means, OR in-progress):
- [ ] Do NOT end the video call if patient is actively in danger
- [ ] Instruct patient to call 911 (provide address if known)
- [ ] Call 911 yourself using patient's address -- identify as treating psychiatrist
- [ ] Contact parent/guardian immediately
- [ ] Stay on video/audio until emergency services arrive if possible
- [ ] Document everything in real time or immediately after
- [ ] Peer consultation call within same day if available

**TELEHEALTH-SPECIFIC EDGE CASES (decision table):**
- Patient disconnects during crisis: Call patient immediately -> call parent/guardian -> if no contact in 5 minutes, call 911 using address on file
- Patient location unknown (different from file): Ask patient to state current address before calling 911; if patient refuses, 911 with last known address and note uncertainty
- Parent refuses emergency services: Document refusal; consult CANRA obligation if abuse suspected; contact peer consultant; follow up with DCFS if needed
- Minor discloses abuse (CANRA trigger): See CANRA quick-reference; do NOT promise confidentiality before disclosure

**CANRA QUICK-REFERENCE (mandatory reporter obligation):**
- Who: Valentina is mandatory reporter under CA Penal Code 11165.7
- Report to: Local law enforcement or DCFS (LA County: 1-800-540-4000)
- Timeline: Verbal report immediately upon reasonable suspicion; written SS 8572 form within 36 hours
- Form: BCIA 8572 (https://oag.ca.gov/sites/all/files/agweb/pdfs/childabuse/ss_8572.pdf)
- Immunity: No civil or criminal liability for good-faith reports
- This is a quick-reference ONLY per locked decision -- Valentina is already familiar with CANRA

**C-SSRS INSTRUMENT REFERENCE:**
- Full name: Columbia Suicide Severity Rating Scale
- Population: Validated for children age 5+; 600+ peer-reviewed studies
- Versions: Screener (6 items, < 3 min) for routine; Lifetime Baseline for first session; Recent version for follow-ups
- Source: cssrs.columbia.edu (free download)
- Administration: Clinician-administered, plain language, no specialized training required

**CRISIS RESOURCES TABLE:**
| Resource | Contact | Scope |
|----------|---------|-------|
| 988 Suicide & Crisis Lifeline | Call or text 988 | Nationwide, CA-specific routing, 24/7 |
| MRSS (Mobile Response) | Via 988 dispatch | Ages 4-18, mobile crisis team |
| 911 | 911 | When patient location known and immediate danger |
| LA County Crisis Line | (800) 854-7771 | LA County residents, 24/7 |
| 988california.org | Online chat | CA statewide web-based |

Include statewide telehealth note: because Valentina serves patients across all CA counties, protocol uses 988 (which auto-routes to local crisis center) as primary resource. Gathering county of residence at intake reduces response time.

**PEER CONSULTATION / BACKUP PROTOCOL RECOMMENDATION:**
Per locked decision (solo practitioner, no formal clinical backup), include a recommendation section:
- Establish a formal peer consultation agreement with 1-2 board-certified psychiatrists
- Recommended: one adult psychiatrist and one child/adolescent specialist
- Purpose: case consultation, crisis support, coverage during PTO
- Document peer consultant contact info in 1Password vault
- Not a referral service -- a collegial consultation relationship

**PATIENT-FACING SAFETY PLAN TEMPLATE (appendix):**
Include the safety plan template from research:
- Warning signs
- Internal coping strategies and safe places
- People I can talk to (2 contacts with phone numbers)
- Professional help: Dr. Park (424) 248-8090, 988, 911
- Things I can do to make my environment safer
- Patient and provider signature lines

**POST-CRISIS PROTOCOL:**
Per locked decision, include full documentation requirements:
- Charting requirements: what to document in Tebra (C-SSRS score, tier determination, actions taken, parent notification log, referrals made, follow-up plan)
- Timeframe: initial documentation within 2 hours of crisis event; full note within 24 hours
- Parent/guardian notification policy: mandatory for all Tier 2+ for minors; document notification time, person contacted, response
- Mandatory follow-up: 24 hours for Tier 2+, 48 hours for Tier 1
- Provider self-care: acknowledge emotional impact, peer debrief recommended, document any peer consultation

Format: Numbered steps with checkboxes. Professional tone. All crisis contacts pre-populated (not blank fields). Written so a covering provider could follow the protocol without prior context.
